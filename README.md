# 穿搭记录小助手

一个美观实用的本地穿搭管理网站,帮助你记录和管理日常穿搭。

## 🎨 设计特色

- **配色方案**: 蓝色主色调 + 白色背景 + 紫色点缀
- **界面风格**: 现代简约,优雅大方
- **响应式设计**: 支持各种屏幕尺寸

## ✨ 功能模块

### 1. 穿搭记录
- 记录每日穿搭照片
- 标注日期、季节、风格、适用场景
- 记录穿搭单品和心得
- 评分系统(1-5星)
- 搜索和筛选功能

### 2. 穿搭灵感库
- 收集和保存穿搭灵感
- 瀑布流展示
- 添加标签分类
- 快速查看详情

### 3. 场景适配
- 按场景分类穿搭
  - 工作/商务
  - 日常休闲
  - 聚会/约会
  - 运动健身
  - 旅行出游

### 4. 统计分析
- 总记录数统计
- 风格偏好分析
- 场景分布图表
- 穿搭趋势展示

## 📁 文件说明

- `outfit-assistant.html` - 主页面文件
- `outfit-style.css` - 样式文件
- `outfit-script.js` - 功能脚本

## 🚀 使用方法

1. 双击打开 `outfit-assistant.html` 文件
2. 点击"添加记录"开始记录你的穿搭
3. 所有数据自动保存在浏览器本地存储中
4. 支持离线使用

## 💾 数据存储

数据通过浏览器的 localStorage 保存在本地,无需服务器。

## 🎯 技术栈

- HTML5
- CSS3 (Tailwind CSS)
- JavaScript (ES6+)
- TDesign 组件库

## 📝 注意事项

- 图片使用 URL 链接方式添加
- 建议使用图床服务保存穿搭照片
- 清除浏览器数据会导致记录丢失,建议定期导出数据
